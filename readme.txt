== Description ==

WP StopSpam ist ein Plugin für das bekannte CMS und Blog-System WordPress. Es verhindert Fake-Registrierungen ganz ohne Captcha, sondern nur mit Sicherheitsfrage und Bot-Techniken.

* Für deutsche Benutzer einfach zu beantwortende Frage (Wortspiel)
* Spam-Bots fallen auf ein zweites, verstecktes Feld herein, damit wird die Registrierung abgewiesen

== Installation ==

Laden Sie einfach die .zip-Datei hoch und aktivieren Sie das Plugin. Alternativ können Sie es auch über das WordPress Plugin-Verzeichnis installieren.

== Frequently Asked Questions ==

= Kann ich die Frage und/oder die Antwort ändern? =

Ja, dazu ist jedoch ein manueller Eingriff in die Datei notwendig. Vielleicht kommt später ein Adminbereich hinzu.

= Sind Copyright-Hinweise vorhanden? =

Nein, unsere Plugins sind komplett werbefrei. Das nervt einfach nur.
